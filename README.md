# capreleaser
simple capacitor project releasing tool.

## installation
```
npm i -g capreleaser
```
## how to
new fix:
```
capreleaser -f
```

new minor:
```
capreleaser -m
```

new major:
```
capreleaser -M
```

help:
```
capreleaser -h
```
